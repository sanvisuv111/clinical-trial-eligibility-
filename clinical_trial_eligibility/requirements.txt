# Clinical Trial Eligibility Engine - Requirements

No external dependencies required. This application uses only Python standard library modules.

## Python Version
- Python 3.8 or higher

## Standard Library Modules Used
- `dataclasses`: For structured data definitions
- `datetime`: For date calculations
- `typing`: For type hints
- `json`: For JSON data handling
- `enum`: For enumeration types
- `functools`: For functional programming utilities
- `unittest`: For testing framework
- `pathlib`: For file path handling

## Installation

1. Ensure Python 3.8+ is installed:
```bash
python --version
```

2. No additional package installation required

3. Verify the project structure:
```bash
cd clinical_trial_eligibility
ls -la src/
ls -la tests/
ls -la data/
```

## Running the Application

### Run Demonstration
```bash
cd clinical_trial_eligibility
python src/main.py
```

### Run Tests
```bash
python -m unittest tests.test_eligibility_engine -v
```

Or with pytest (optional):
```bash
pip install pytest
pytest tests/test_eligibility_engine.py -v
```
